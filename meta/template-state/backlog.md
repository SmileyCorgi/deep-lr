# Ideas backlog

Prioritized tracker of forward-looking research questions. This is stage 1 of
the idea lifecycle (CLAUDE.md §3): **capture** here cheaply → **shape** with
the `good-question` skill when it gets serious → **promote** to an experiment
or deep-dive when the trigger fires → **prune** during lint when superseded.

Entry format (parsed by `portal/build.py` — keep the field names exact):

```
## <one-line idea title>
- Date: YYYY-MM-DD
- Trigger: what would make this worth starting
- Sketch: 2-line sketch; after shaping, link the good-question card / evidence
- Promoted to: (empty until promoted — then experiments/active/<slug> or the synthesis page)
```
